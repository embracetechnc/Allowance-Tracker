## Setup

### Weekly Recurring Tasks

Each task is worth 5 points and recurs weekly:

1. **Clean Room**

   - Make bed
   - Pick up clothes
   - Vacuum floor
   - Dust surfaces

2. **Clean Bathroom**

   - Clean toilet
   - Clean sink
   - Clean shower/tub
   - Mop floor
   - Replace towels

3. **Clean Kitchen**

   - Wipe counters
   - Clean appliances
   - Sweep/mop floor

4. **Laundry**
   - Wash one load
   - Dry clothes
   - Fold clothes
   - Put away clothes

## Child Instructions

### Viewing Tasks

- Log in to your dashboard
- View assigned tasks
- Each task shows:
  - Task name
  - Description
  - Points value (5 points)
  - Due date
  - Current status

### Task Status Colors

- Gray: Assigned
- Yellow: In Progress
- Blue: Completed (waiting for verification)
- Green: Verified
- Red: Rejected

### Completing Tasks

1. **Starting a Task**

   - Click "Start" when beginning
   - Task status changes to "in progress"

2. **Marking Complete**
   - Click "Complete" when finished
   - Task turns blue
   - Parent is notified for verification

## Parent Instructions

### Task Verification

1. **View Completed Tasks**

   - Check Parent Dashboard
   - Look for blue "completed" tasks
   - Review:
     - Task name
     - Child who completed it
     - Completion time
     - Points value

2. **Verification Options**
   - Click "Verify" if done correctly
     - Task turns green
     - 5 points awarded
     - Added to history
   - Click "Reject" if needs work
     - Add note explaining what to fix
     - Task returns to child's list
     - No points awarded until fixed and verified

### Managing Allowance

1. **Calculate Points**

   - Click "Manage Allowance" on child's card
   - Select date range (typically weekly)
   - System calculates:
     - Total verified tasks
     - Points earned
     - Allowance amount

2. **Weekly Review**
   - Tasks reset weekly
   - Review:
     - Completed tasks
     - Points earned
     - Allowance history

## Points System

- Each task = 5 points
- Points awarded only after verification
- Weekly tasks = potential 20 points per week
- Points tracked:
  - Per task
  - Per week
  - Total accumulated

## Weekly Schedule

All tasks should be completed by Sunday evening for weekly verification and allowance calculation.

### Task Schedule Example

- Monday-Sunday: Complete assigned tasks
- Sunday evening: Parent verification
- Monday: New tasks assigned

## Tips for Success

1. **For Children**

   - Start tasks early in the week
   - Follow task descriptions carefully
   - Mark tasks complete only when fully finished
   - Check rejected tasks for parent notes

2. **For Parents**
   - Verify tasks promptly
   - Provide clear feedback if rejecting
   - Review weekly totals
   - Maintain consistent schedule

## Troubleshooting

If you encounter issues:

1. Check task status
2. Verify completion requirements
3. Review point calculations
4. Contact support if needed
